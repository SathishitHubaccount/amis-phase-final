
## SLIDE 8: Safety & Control - Human-in-the-Loop

```
┌────────────────────────────────────────────────────────────┐
│  "What if AI makes a mistake?"                             │
│                                                            │
│  4-LEVEL RISK-BASED APPROVAL SYSTEM                        │
│                                                            │
│  🟢 LOW RISK - Auto-Approved                               │
│     • Forecasts, analytics, reports                        │
│     • Financial impact: < $1,000                           │
│     • Example: "Update demand forecast"                    │
│     → Executed automatically, logged for audit             │
│                                                            │
│  🟡 MEDIUM RISK - Manager Review                           │
│     • Inventory adjustments, minor schedule changes        │
│     • Financial impact: $1,000 - $10,000                   │
│     • Example: "Order 500 units for $5,000"                │
│     → Manager notified, can approve/reject in AMIS         │
│                                                            │
│  🟠 HIGH RISK - Manager Approval Required                  │
│     • Production schedule changes, overtime                │
│     • Financial impact: $10,000 - $100,000                 │
│     • Example: "Schedule overtime for Week 3"              │
│     → Manager must explicitly approve before execution     │
│                                                            │
│  🔴 CRITICAL RISK - Multi-Level Approval                   │
│     • Major investments, safety concerns, plant shutdowns  │
│     • Financial impact: > $100,000 or safety risk          │
│     • Example: "Shutdown MCH-004 for emergency repair"     │
│     → VP + Manager + Safety Officer approval required      │
│                                                            │
│  ✅ COMPLETE AUDIT TRAIL                                   │
│     • Every AI decision logged                             │
│     • Who approved, when, why                              │
│     • Rollback capability                                  │
│     • Compliance ready                                     │
└────────────────────────────────────────────────────────────┘
```

**Speaker Notes:**
- Most common concern: "What if AI messes up?"
- We have comprehensive safety controls
- Low-risk items auto-approved (saves time)
- High-risk items require human judgment
- Critical items require multiple approvals
- Complete transparency - nothing hidden
- Can always override or rollback
